# Vagi SSO Based on CAS

###Version 1.0 GA

Blog : http://shimingxy.blog.163.com/

Welcome to the home of  the  Vagi project, Vagi SSO Based on CAS.  

Vagi是一款基于<strong>CAS</strong>(cas web https://github.com/Jasig/cas)的Web应用单点登录系统.

- 对数据库用户存储支持
- 添加登录的验证码
- 新浪微博和QQ互联的支持
- 历史登录的日志
- 记住登录用户名
- 记住登录
